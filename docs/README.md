# 个人导航主页应用

一个现代化的个人导航主页，提供主题切换、企业链接管理、书签分类、快捷链接、待办事项等功能。

## 项目特点

### 🎨 主题切换
- 支持明亮/暗黑模式
- 自动适配系统主题偏好
- 跨标签页主题同步

### 🏢 企业链接
- 集中管理企业内部系统
- 支持密码锁定保护
- 网格布局，一目了然

### 📚 书签管理
- 分类管理个人书签
- 支持隐藏/显示特定分类
- AI智能分类（预留功能）

### ⚡ 快捷链接
- 自定义常用网站入口

### ✅ 待办事项
- 集成待办事项面板
- 支持添加/删除/完成任务
- 本地数据持久化

### 🔍 搜索功能
- 支持多种搜索引擎切换
- 快速搜索入口

### 📅 日历集成
- 显示今日事件提醒

### 🖼️ 背景自定义
- 支持上传自定义背景图片
- 预设背景图片选择
